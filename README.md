## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jimblu/WikiPopulation_CLI-Web-Scraper.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Use it
   ```sh
   npm run popWiki
   ```
